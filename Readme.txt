To introduce a linebreak, add a <br> tag in the rich text editor.\

To use bold style, wrap the line in a b tag

Example: <b>Text within is bold<b>

To use italics, wrap the line in an i tag

Example: <i>Italicized Text Here</i>

To use underlined 
wrap the line in a u tag

Example: <p>This paragraph contains some <u>mispelled</u> text.</p>
